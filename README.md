# 📊 Assignment 2: Sentiment Analysis on Pokémon Tweets

---

## (1) Problem Statement

The objective of this project is to perform sentiment analysis on Pokémon-related tweets and classify them into positive, neutral, and negative categories using machine learning techniques.

---

## (2) Objective

* To preprocess tweet data
* To convert text into numerical format
* To train multiple classification models
* To compare model performance and select the best one

---

## (3) Dataset

**Source:**
Synthetic Pokémon-related tweets generated for academic use

**Features:**

* Tweet (text data)
* Sentiment (positive / neutral / negative)

**Size:**

* 100 tweets

## (4) Methodology
